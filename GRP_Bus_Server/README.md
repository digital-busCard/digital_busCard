## Tech Stack

**Server:**
```
- Java 11
- Springboot 2.6.6
- Gradle 7.4.1
```
**Database:**
```
- Postgres 13.1 alpine
```

**Run Local**

To run locally, please install necessary components becore running the command line below.
```
./gradlew clean build
docker compose up
```
